# Alumni Project Space @Msannu
## Basic Information
### What is this site?
Alumni Project Space @Msannu or DBSDF Zone is a platform which established by alumni of Msannu.Here, according to the desire of the project authors, a display space will be provided for projects related to Msanu or made by alumni of Msannu at Github.
### What can I do?
If you want your project to appear on our list, or if you want to contribute to this project, please don't worry. Our page will be officially launched after internal testing. Please stay tuned to this website for more information.
### Who is behind it?
The team from Spride is building and maintaining this project. Do you want to learn more?
